

True or False
The simplest Well-Formed Formula (wff) in [[Propositional Logic]]
Not able to be split into smaller wff

|       |       |        |
| ----- | ----- | ------ |
| True  | **t** | $\top$ |
| False | **f** | $\bot$ |

Atoms and negations of atoms are called **literals**
- positive literal (e.g. $x$)
- negative literal (e.g. $\neg x$)
